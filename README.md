# Pinterest Matblogg

Enkel, statisk blogg for engelske matoppskrifter.

## Status
- HTML/CSS-blogg deployes via Netlify fra GitHub-repo `Nettsam1-Hermes/pinterest-matblogg`
- Raspener: 3–5 oppskrifter per uke, jevnlig planleggings-cron i Obsidian
- Bildene trenger å lages og lagres i `assets/` per oppskrift

## Deployment
1. Gå til Netlify → "New site from Git"
2. Velg GitHub og repoet `Nettsam1-Hermes/pinterest-matblogg`
3. Build settings: Publish directory = `.`, Command = tom
4. Netlify vil automatisk bygge og publisere ved hver push til `main`

## Oppslagsverk
- `01 Oppslagsverk/Pinterest matblogg.md`: research og strategi
- `03 Prosjekter/Pinterest matkanal/`: innholdsplan og oppskriftsnotater
